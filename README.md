# 🚴‍♂️ Bike Sales Analysis Dashboard

## 📊 Project Overview
This project presents an interactive dashboard designed to analyze bike sales data and uncover key customer trends. The goal of this analysis is to identify factors that influence purchasing behavior and support data-driven decision-making.

---

## 🛠️ Tools & Technologies
- Microsoft Excel  
- Power BI  

---

## 📌 Problem Statement
Understanding customer behavior is essential for improving sales performance. This project explores how demographic and lifestyle factors such as income, age, and commute distance influence bike purchases.

---

## 📈 Key Insights
- Customers with higher income levels are more likely to purchase bikes  
- Middle-aged individuals represent the highest number of buyers  
- Shorter commute distances are associated with higher purchase rates  
- Certain regions show stronger sales performance than others  

---

## 🖥️ Dashboard Features
- Interactive filters for dynamic data exploration  
- Visual breakdown of customer demographics  
- Clear representation of sales trends and patterns  
- User-friendly and intuitive layout
